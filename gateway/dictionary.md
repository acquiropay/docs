## Состояния платежа

| Название   | Описание                      |
|------------|-------------------------------|
| OK         | Платеж успешно завершен.      |
| KO         | Платеж не прошел.             |
| PROCESSING | Платеж находится в обработке. |
| 3DSECURE   | Требуется 3DS авторизация.    |
| CANCEL     | Платеж отменен.               |
| CHARGEBACK | Возврат платежа.              |
